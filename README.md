1.下载安装包‌：访问[Node.js官网](https:nodejs.org)，选择带“LTS”（长期支持）标识的版本下载。‌

2.安装electron,vite

```sh
npm install electron vite vite-plugin-electron --save-dev
npm install @electron/rebuild --save-dev
```

3.运行

启动第一个客户端并运行
```sh
npm run dev
```
启动第二个客户端并运行
```sh
npm run build
```
